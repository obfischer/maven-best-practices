= Maven Best Practices Guidelines
:description: Common best practices for using Apache Maven 3 in real projects

This guide summarizes my personal best practices I have collected and developed in using Apache Maven 3 in real projects.
I decided to put them all together to be able to reference them and to easily share them with colleagues and friends, without repeating myself over and over.
Now I am able to share them via a simple hyperlink.

I hope, dear reader, if you find them, you can find something useful for you.


NOTE: Currently, this guide covers only Apache Maven 3.
Maven 4 is currently under development and hasn't been released yet.
As soon as I have gathered enough knowledge about Maven 4, I will update and extend this guide.
